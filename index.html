BlogPost API

GitHub Repository

A simple RESTful API built using Django REST Framework (DRF) to perform basic CRUD (Create, Read, Update, Delete) operations for blog posts.

Core Features:
  1. Create a Blog Post:

    Allows users to create new blog posts by providing a title, content, and metadata like author and publication date.

  2. Read Blog Posts:

    Fetch all blog posts or retrieve a specific post using its unique identifier. Supports pagination for better handling of large datasets.

  3. Update a Blog Post:

    Update the title, content, or other details of an existing blog post.

  4. Delete a Blog Post:

    Permanently delete a blog post from the database.

Tech Stack:
  • Backend Framework: Django REST Framework
  • Database: SQLite (configurable)
Endpoints
    • POST /api/blogposts/ - Create a new blog post (Handled by BlogPostCreate)
    • GET /api/blogposts/<id>/ - Retrieve a specific blog post (Handled by BlogPostRetrieveUpdateDestroy)
    • PUT /api/blogposts/<id>/ - Update a specific blog post (Handled by BlogPostRetrieveUpdateDestroy)
    • DELETE /api/blogposts/<id>/ - Delete a specific blog post (Handled by BlogPostRetrieveUpdateDestroy)
